Workspace sidebar

Aesthetic notes
  • Fonts match the existing app — system sans for chrome, JetBrains Mono only where the app already uses it (project paths, count chips, session timestamps).
  • Workspace names: uppercase tracked sans, weight 600 — sits as a header tier above project rows without introducing a new typeface.
  • Each workspace owns a 3px color spine (--workspace-color-1..6) that runs the full height of the expanded body. Collapsed workspaces shrink it to a 12px nub — the rail visibly retracts.
  • "Ungrouped" gets a dotted spine and muted tracked-caps name — it's a holding bay, not a peer.
  • Drag-target workspaces (.drag-target on "Open Source") get an --accent-subtle wash + dashed outline.
  • Filter input lives between masthead and the workspace stack. ⌘/ kbd hint mirrors xterm ⌘F.
  • Three theme palettes; workspace colors are retuned per theme for legibility (a saturated red on translucent looks washed out without a brighter token).
