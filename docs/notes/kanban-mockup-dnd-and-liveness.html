What's new in this mockup

  • Drag-and-drop column moves — pointer-down on a tile starts a drag; the destination column shows an accent border and a dashed placeholder where the card will land. On drop, fires kanbanStore.move(id, status).
  • Live session pulse — running (green, animating), idle (gray, static), error (red). Tile subscribes to agent-store via the linked conversationId.
  • Unread badge — turns completed since the user last viewed the chat. Cleared when user opens the conversation.
  • Cost progress bar — visualizes costUsedUsd / costCapUsd. Switches to amber at 75%, red at 100%.
  • "Needs input" auto-promotion — when the runtime emits question.asked for a card's session, the card flips status and gets the amber halo + "awaiting reply" ribbon. On question.answered, demote back to in_progress.
  • Relative time — "12s", "4m", "completed 1h ago". Uses existing fmtDuration.
